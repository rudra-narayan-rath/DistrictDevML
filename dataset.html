Dataset Description

This dataset contains development indicators for 10 Indian districts across four sectors: Water Supply, Healthcare, Road Infrastructure, and Education.

Each record includes a region name, sector, indicator description, and a numerical value.

The machine learning model relies entirely on this dataset to compute priority scores.
